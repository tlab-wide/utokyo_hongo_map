# UTokyo Hongo Campus Point Cloud Map

A 3D **point cloud** covering the entire **~4 km road network** of the University of
Tokyo's **Hongo campus**, scanned in **September 2015** and published as open data.

## 📥 Get the data

Request access here — enter a little information about yourself and your intended use,
and the download links will appear:

**https://tlab-wide.github.io/utokyo_hongo_map/**

## About

The point cloud data were scanned in **September 2015** as part of the Graduate Program
for Social ICT Global Creative Leaders (GCL), using the mobile mapping system of
**Aisan Technology Co., Ltd.** The data cover the **~4 km road network of the entire
Hongo campus** of the University of Tokyo. Published **August 2020**.

**Available formats:** colored point cloud (PCD, full & partial), uncolored point cloud
(PCD, 5 cm and 20 cm downsampled), and uncolored point cloud (e57).
